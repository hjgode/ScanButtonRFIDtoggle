========================================================================
    CONSOLE APPLICATION : ScanButtonRFIDtoggle Project Overview
========================================================================

AppWizard has created this ScanButtonRFIDtoggle application for you.  

ScanButtonRFIDtoggle will change the center scan button to RFID or SCAN

if launched with no args, the center scan button is toggled between RFID and SCAN

if launched with RFID, the center scan button will be mapped to RFID

if launched with SCAN, the center scan button will be mapped to Barcode Scanner
